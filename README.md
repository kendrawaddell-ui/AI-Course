# AI-Course
An 8-unit course designed to teach computer programmers how code beside and to integrate systems with AI. To complete the course, students should be proficient in either Python, JavaScript, or C#. They will also need  Github Copilot subscription for the entireity of the course. Units include: 

1. Foundations of Generative AI  
2. Prompt Engineering Fundamentals
3. AI Orchestration
4. Retrieval Augmented Generation (RAG)
5. Responsible Use of AI & Ethical Considerations
6. MCP & Advanced Function Calling
7. Agentic AI Systems
8. Capstone Project
